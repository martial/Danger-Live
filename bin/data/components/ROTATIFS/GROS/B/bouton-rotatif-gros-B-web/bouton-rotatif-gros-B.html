Vous avez exporté bouton-rotatif-gros-B à l’aide de QuickTime Player. Les informations qui suivent vous aideront à publier cette séquence sur votre site web.

Dans le même dossier que le présent fichier HTML d’instructions, QuickTime Player a placé différentes versions de votre séquence optimisées pour la publication sur le web. En publiant aussi cette séquence de référence dans votre page HTML, vous pouvez vous assurer que les visiteurs visionneront toujours, sur votre site, la vidéo de la qualité idéale.

Remarques importantes :

  • Pour que vos données puissent être visionnées sur Internet, vos fichiers de données (séquences, affiche) doivent être placés sur un serveur accessible publiquement.
  • Les paramètres d’URL doivent être modifiés si vos données sont servies à partir d’un autre emplacement que votre page web.
  • Les paramètres d’URL de votre séquence doivent toujours pointer vers une séquence de référence (bouton-rotatif-gros-B.mov) plutôt que vers une séquence source.
  • La séquence de référence doit se trouver dans le même dossier que les séquences source.
  • Les noms des séquences source ne peuvent pas être modifiés, mais vous pouvez modifier le nom de la séquence de référence (n’oubliez pas de le modifier aussi dans le code HTML).
  • Pour être validés correctement comme du XHTML, la balise <style> et la première balise <script> doivent être placées dans la balise <head> de votre page.
